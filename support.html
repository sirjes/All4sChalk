Frequently Asked Questions

How do I track points?

Use the + and - buttons next to each team's score to add or subtract points. For game points when "Game Worth 2" is enabled, use the "Game" button to add 2 points at once.

Can I customize team names?

Yes! Simply tap on the team name to edit it. Your custom team names will be saved for future games.

How do I undo a move?

Use the undo button (curved arrow) in the game controls to reverse your last move. You can undo multiple moves in sequence.

What is "Game Worth 2"?

This option allows you to toggle between regular scoring and double points for game points. When enabled, using the "Game" button will add 2 points instead of 1.

How do I view game history?

Tap the history button to view a list of all moves in the current game. The history shows each move and the resulting score.

How do I share game results?

Use the share button (square with arrow) to capture and share the current game state with others.
